# Yugo2Market Clean

A fresh, deployable monorepo for Yugo2Market:

- `app/` = Vite + React frontend for Vercel
- `backend/` = Express + MongoDB API for Railway

## Deploy

- Vercel root directory: `app`
- Railway root directory: `backend`

## Local env

Copy the `.env.example` files in `app/` and `backend/`.
